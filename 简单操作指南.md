##操作指南
1.从master分支克隆项目

git clone git@github.com:CSLiuPeng/NewHardware_DBMS_PaperList.git

2.查看分支

git branch

3.切换到Dev分支，在平时开发过程中master作为主要分支，一般不直接在master上开发

git checkout dev

4.在开发分支上拉取代码

git pull

5.提交Dev分支上的代码

git add .

git commit -m "操作说明"

6.将代码提交到远程仓库

git push origin dev

7.将Dev分支上的代码合并到master分支

git merge dev

8.推送到将Dev分支代码推送到master分支

git push origin master
